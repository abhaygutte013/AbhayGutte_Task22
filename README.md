# Project Workspace

## About

This is a simple Project Workspace landing page made using HTML and Tailwind CSS.

## Features

- Sticky Navbar
- Hero Section
- Gradient Background
- Two Buttons
- Workspace Image
- Floating Cards
- Fixed Chat Button

## Tailwind CSS Classes Used

- flex
- relative
- absolute
- fixed
- sticky
- top-0
- bottom-6
- right-6
- z-20
- z-30
- z-50
- shadow
- rounded

- How to create a responsive layout
